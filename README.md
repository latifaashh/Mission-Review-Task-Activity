# Monitoring Submit Dashboard

Dashboard Python berbasis Streamlit untuk monitoring submit dan no submit activity berdasarkan cycle.

## Features

* Upload file Excel / CSV
* Filter berdasarkan Cycle
* Tabel Summary Submit vs No Submit
* Ranking activity tidak submit
* Download hasil summary Excel

## Install Library

```bash
pip install -r requirements.txt
```

## Menjalankan Dashboard

```bash
streamlit run app.py
```
